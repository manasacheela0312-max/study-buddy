An AI-powered study roadmap generator for students who don't know where to start.
What it does

Type any topic → get a structured week-by-week study roadmap
Upload your course syllabus (PDF) → get a roadmap based on your actual curriculum
80/20 focus list — the 20% of concepts that give you 80% of the results

Tech Stack

HTML, CSS, JavaScript (no frameworks)
Groq API — LLaMA 3.3 70B for roadmap generation
PDF.js — in-browser PDF text extraction (no file upload to any server)
